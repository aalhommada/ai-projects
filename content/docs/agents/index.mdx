# AI Agents

Create AI systems that can plan, reason, and take actions. From simple tool-using agents to complex multi-agent systems.

| [ReAct Agent](/docs/agents/basic/react-agent) | Reasoning and acting pattern | ~3 hours |

### Advanced

| Project | Description | Time |

## Key Concepts

- Agent architectures (ReAct, Plan-and-Execute)
- Tool definition and calling
- Memory systems (short-term, long-term)
- Multi-agent coordination
- Safety and guardrails
